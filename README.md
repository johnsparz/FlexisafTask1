# Customer Churn Prediction using Ensemble Learning & Hyperparameter Optimization

This project focuses on predicting customer churn using advanced machine learning techniques such as Ensemble Learning and Hyperparameter Optimization.

The goal was to build and evaluate machine learning models capable of identifying customers likely to leave a telecom service provider based on customer demographics, account information, and service usage patterns.

---

# Techniques Implemented

## Ensemble Learning
The following ensemble models were trained and evaluated:
- Random Forest Classifier
- Gradient Boosting Classifier

These models were used to improve prediction accuracy, reduce overfitting, and increase model robustness compared to a baseline Decision Tree model.

---

## Hyperparameter Optimization
Hyperparameter tuning was performed using:

```python
GridSearchCV

```

The optimization process improved model performance by identifying the best combination of parameters for the Random Forest model.
